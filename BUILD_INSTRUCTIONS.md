# Build Instructions

## Prerequisites

- Node.js
- NPM

## Build

```bash
npm install
npm build
```

Typescript

```bash
tsc build tsc
```

Enable this features in chrome://flags

```
--enable-summarization-api-for-gemini-nano
--enable-wrtier-api-for-gemini-nano
--enable-rewriter-api-for-gemini-nano
```

Load unpacked extension
